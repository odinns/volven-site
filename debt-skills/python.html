Python skill

Find the shape before runtime does.

Use this skill for Python code where the expensive part is usually the missing boundary, not the missing formatter run.

Signals

  • Dictionaries and JSON payloads crossing core paths without schema checks.
  • Mutable module state that tests and jobs accidentally share.
  • Swallowed exceptions that turn failures into stale data.
  • Async code without cancellation, timeout, or retry discipline.
  • Scripts that became production workflows without ownership or tests.

Boundary

The skill does not replace Ruff, mypy, pyright, Black, isort, or normal test failures. It reviews the contracts those tools can only hint at.

Best Use

Use it on API clients, data pipelines, async workers, CLI scripts, import jobs, and code that glues external systems together.
