PHP skill

Find the PHP contracts hiding in plain sight.

Use this skill after Pint, PHPStan, Psalm, Rector, and the test suite have done their normal jobs.

Signals

  • Array shapes passed through core paths without a named contract.
  • `mixed` at boundaries where the shape is used before validation.
  • Magic workflow strings that should be enums or value objects.
  • PHPDoc that silences uncertainty without making behavior clearer.
  • Tests that mock internals while missing real boundary behavior.

Boundary

The skill does not replace static analysis. It asks what still gets expensive after static analysis is green.

Pair It With Pest

The Pest plugin gives PHP projects a deterministic ratchet for suppressions, TODOs, `mixed`, and large classes.
